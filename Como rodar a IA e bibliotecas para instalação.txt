1. bibliotecas

Primeiro é preciso instalar as seguintes bibliotecas:

Flask → framework web em Python.

flask_cors (CORS) → extensão para habilitar Cross-Origin Resource Sharing.

google.generativeai → SDK da Google para usar modelos de IA generativa.

os → biblioteca padrão do Python para interagir com o sistema operacional.

dotenv (load_dotenv) → carrega variáveis de ambiente de um arquivo .env

Pode usar o seguinte comando:
pip install flask flask-cors google-generativeai python-dotenv
